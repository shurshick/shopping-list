# Список покупок v1.0.0

Публичная версия приложения для Android и серверной части для собственного TrueNAS/Docker-сервера.

## Главное

- Android-приложение для совместных списков покупок.
- Сервер синхронизации на FastAPI и PostgreSQL.
- Развертывание через TrueNAS 25.04 Custom App, обычный Docker Compose, сборку напрямую из GitHub или готовый GHCR-образ.
- Первичная настройка сервера через веб-мастер `/setup`.
- Обязательное создание администратора при первом запуске.
- APK подписывается постоянным ключом проекта, поэтому обновления после `v0.2.3` устанавливаются поверх.

## Новое в версии 1.0.0

- Добавлена очистка только купленных товаров.
- Добавлено редактирование товара: можно изменить название и количество.
- Добавлен экран настроек Android-приложения с адресом сервера и выходом из аккаунта.
- В основное меню добавлен пункт "О приложении" с версией, копирайтом и ссылкой на GitHub-проект.
- Купленные товары остаются внизу списка, а возвращенные в покупки товары поднимаются наверх.
- Обновлены README, API-документация и инструкции.

## Возможности приложения

- Регистрация и вход пользователей.
- Несколько списков покупок.
- Совместный доступ по email.
- Одноразовые ссылки-приглашения.
- Просмотр участников списка.
- Переименование, копирование, очистка и удаление списков.
- Удаление списка у участника отключает только его доступ.
- Добавление, редактирование, отметка и удаление товаров.
- Справочник товаров и автопоиск при вводе.
- Синхронизация при запуске и после действий пользователя.

## Важное примечание по обновлению APK

Если на телефоне установлена версия `v0.2.2` или ниже, Android может не поставить `v1.0.0` поверх из-за старой временной подписи APK. В этом случае удалите старую версию один раз и установите `v1.0.0`.

Если уже установлена `v0.2.3` или новее, обновление должно установиться поверх.

## Файлы релиза

- `shopping-list-android-v1.0.0.apk` - Android-приложение для установки на телефон.
- `shopping-list-server-v1.0.0.zip` - серверная часть, Docker Compose-файлы и инструкции.
- `shopping-list-source-v1.0.0.zip` - полный архив исходного кода.
