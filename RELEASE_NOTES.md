# Список покупок v1.0.1

Обновление имени проекта, описаний и документации развертывания.

## Что изменено

- Проект переименован в `shopping-list`, без привязки имени к TrueNAS.
- TrueNAS теперь описан как один из вариантов развертывания, а не как обязательная платформа.
- Обновлены ссылки на GitHub-проект.
- Обновлено имя Docker-образа GHCR: `ghcr.io/shurshick/shopping-list-api`.
- Обновлены Docker Compose-файлы для сборки backend из нового репозитория.
- В README и API-документацию добавлены требования Android.
- В окне "О приложении" ссылка теперь ведет на новый GitHub-проект.

## Поддержка Android

- Минимальная версия Android: Android 8.0 Oreo.
- Минимальный API: 26.
- Целевая версия сборки: Android 15, API 35.

## Способы запуска сервера

Сервер можно запускать не только на TrueNAS:

- обычный Docker Compose на любом сервере;
- сборка backend напрямую из GitHub;
- готовый GHCR-образ;
- TrueNAS 25.04 Custom App через YAML.

## Файлы релиза

- `shopping-list-android-v1.0.1.apk` - Android-приложение для установки на телефон.
- `shopping-list-server-v1.0.1.zip` - серверная часть, Docker Compose-файлы и инструкции.
- `shopping-list-source-v1.0.1.zip` - полный архив исходного кода.
