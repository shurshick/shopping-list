# Список покупок v1.5.0

Релиз `v1.5.0` улучшает ежедневную работу со списками без утяжеления приложения.

## Изменения этого релиза

- Товар можно быстро добавить через Done/Enter на клавиатуре.
- После добавления поле товара очищается, остаётся в фокусе и готово к следующему вводу.
- Кнопка `+` рядом с полем добавления сохранена.
- Добавлена кнопка очистки поля товара, которая не сбрасывает фокус.
- Пустой товар не добавляется, лишние пробелы обрезаются.
- Подсказки товаров остались компактными и работают по локальному справочнику.
- Улучшен выбор списков: текущий список виден, при большом количестве списков доступен поиск.
- Добавлен локальный основной список: он открывается первым после запуска, если доступен.
- Создание списка trim-ит название, запрещает пустое имя и фокусирует поле ввода.
- Улучшены пустые состояния для пустого списка, отсутствующих списков и ситуации, когда всё куплено.
- Offline queue и `client_operation_id` не менялись и продолжают работать как раньше.

## Совместимость

- Backend API не расширялся.
- Новых тяжёлых зависимостей нет.
- Room, WebSocket/SSE и push-уведомления не добавлялись.
