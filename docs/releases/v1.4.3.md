# Список покупок v1.4.3

Публичный релиз Android-приложения и серверной части для самостоятельного размещения.

## Что изменилось в этом релизе

- PostgreSQL healthcheck в Docker Compose изменён с 10 секунд на 30 секунд.
- Страница `/admin` теперь отдаётся с запретом кеширования, чтобы браузер не показывал устаревший интерфейс администрирования.
- В админке пользователей появились данные Android-клиента: приложение, версия, `versionCode`, платформа и последний запуск.
- Версии обновлены до `API 1.4.3` и `Android 1.4.3` (`versionCode = 26`).

## Совместимость

- Android: 8.0 Oreo и новее, `minSdk = 26`.
- Docker image: `ghcr.io/shurshick/shopping-list-api:latest` и `ghcr.io/shurshick/shopping-list-api:v1.4.3`.

## Файлы релиза

- `shopping-list-android-v1.4.3.apk`
- `shopping-list-server-v1.4.3.zip`
- `shopping-list-source-v1.4.3.zip`

## Обновление сервера

1. Сделайте backup БД или выполните `python -m app.cli backup --output backup.json`.
2. Обновите Docker image до `latest` или `v1.4.3`.
3. Перезапустите контейнер API.
4. Проверьте `/health/ready` и `/admin/system`.
