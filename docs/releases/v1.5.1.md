# Список покупок v1.5.1

Небольшой релиз с исправлениями ежедневной работы со списками.

## Изменения этого релиза

- Справочник товаров теперь очищает уже сохранённые дубли с разным регистром и лишними пробелами.
- Добавление товара больше не переключает пользователя на основной список: текущий список сохраняется при обычной синхронизации.
- Основной список по-прежнему открывается первым только после запуска приложения или нового входа.
