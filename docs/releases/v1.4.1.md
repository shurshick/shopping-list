# Список покупок v1.4.1

Публичный релиз Android-приложения и серверной части для самостоятельного размещения.

## Что изменилось в этом релизе

- Новые admin/ops разделы теперь доступны прямо из веб-интерфейса `/admin`, без ручного ввода URL.
- Добавлен блок "Администрирование" с кнопками: пользователи, списки, приглашения, система, логи, диагностика.
- В `/admin` добавлены таблицы пользователей, списков и приглашений.
- Из `/admin` можно отключить или включить пользователя, задать пароль, архивировать или восстановить список и отозвать приглашение.
- Добавлен тест, который проверяет наличие новых admin-разделов на странице `/admin`.
- Версии обновлены до `API 1.4.1` и `Android 1.4.1` (`versionCode = 24`).

## Совместимость

- Android: 8.0 Oreo и новее, `minSdk = 26`.
- Docker image: `ghcr.io/shurshick/shopping-list-api:latest` и `ghcr.io/shurshick/shopping-list-api:v1.4.1`.

## Файлы релиза

- `shopping-list-android-v1.4.1.apk`
- `shopping-list-server-v1.4.1.zip`
- `shopping-list-source-v1.4.1.zip`

## Обновление сервера

1. Сделайте backup БД или выполните `python -m app.cli backup --output backup.json`.
2. Обновите Docker image до `latest` или `v1.4.1`.
3. Перезапустите контейнер API.
4. Проверьте `/health/ready` и `/admin/system`.
