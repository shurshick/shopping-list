# Список покупок v1.5.3

Релиз приводит Android-интерфейс и процесс сборки в порядок без изменения API и базы данных.

## Изменения этого релиза

- Настройка `Объединять одинаковые товары` убрана с главного экрана и перенесена в меню списка под кнопкой-шестерёнкой.
- Настройка осталась чекбоксом и продолжает сохраняться между запусками приложения.
- Убраны устаревшие варианты Compose-иконок.
- Веб-админка получила тёмный Material-подобный дизайн, плиточную навигацию и адаптивные таблицы для телефона.
- После входа форма авторизации скрывается и не занимает место в рабочем интерфейсе.
- Вывод данных в статусных плитках защищён от HTML-инъекций.
- Backend переведён с устаревших startup hooks и `datetime.utcnow()` на актуальные API без изменения формата данных в БД.
- Удалён дублированный импорт в административном API.
- Android CI теперь запускает unit-тесты и lint перед сборкой debug APK.
- Ключ подписи APK и его параметры удалены из актуального дерева проекта и перенесены в GitHub Actions Secrets.
- Release workflow проверяет наличие секретов и валидность подписи собранного APK.
