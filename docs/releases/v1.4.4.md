# Список покупок v1.4.4

Публичный релиз Android-приложения и серверной части для самостоятельного размещения.

## Что изменилось в этом релизе

- Исправлено отображение платформы Android-клиента в `/admin/users`: вместо технического фрагмента шаблона показывается нормальное значение, например `android 14`.
- Проверка обновлений Android при запуске больше не блокируется 12-часовым интервалом: при каждом запуске приложение проверяет последний GitHub Release.
- При недоступности GitHub или сети уведомление не показывается, приложение продолжает работать без ошибок.
- Версии обновлены до `API 1.4.4` и `Android 1.4.4` (`versionCode = 27`).

## Совместимость

- Android: 8.0 Oreo и новее, `minSdk = 26`.
- Docker image: `ghcr.io/shurshick/shopping-list-api:latest` и `ghcr.io/shurshick/shopping-list-api:v1.4.4`.

## Файлы релиза

- `shopping-list-android-v1.4.4.apk`
- `shopping-list-server-v1.4.4.zip`
- `shopping-list-source-v1.4.4.zip`

## Обновление сервера

1. Сделайте backup БД или выполните `python -m app.cli backup --output backup.json`.
2. Обновите Docker image до `latest` или `v1.4.4`.
3. Перезапустите контейнер API.
4. Проверьте `/health/ready` и `/admin/system`.
