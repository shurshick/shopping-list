# Release Notes Files

В этом каталоге лежат отдельные тексты для GitHub Releases.

Один тег GitHub Release = один файл:

- `v1.4.6` -> `docs/releases/v1.4.6.md`
- `v1.4.7` -> `docs/releases/v1.4.7.md`
- `v1.4.5` -> `docs/releases/v1.4.5.md`

Это позволяет держать описание каждого релиза коротким и не включать в него заметки от предыдущих версий.
