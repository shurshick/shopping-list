# Список покупок v1.4.7

Публичный релиз Android-приложения и серверной части для самостоятельного размещения.

## Что изменилось в этом релизе

- Исправлена безопасная смена сервера в Android с явной моделью `ServerSettings`.
- Тестовый сервер `https://rust.bghitech.ru` доступен на экране входа и регистрации через чек-бокс `Использовать тестовый сервер`.
- В настройках Android теперь показываются текущий сервер, тип сервера и кнопка `Сменить сервер`, без мгновенного переключения внутри активной сессии.
- Смена сервера требует подтверждения и завершает текущую сессию, что предотвращает использование token от одного сервера на другом.
- При подтвержденной смене сервера очищаются токен, выбранный список, локальный кэш списков и pending offline operations.
- Ручной сервер сохраняется отдельно и не теряется при включении и отключении тестового сервера.
- Добавлены Android unit-тесты для `ServerSettings` и cleanup состояния при смене сервера.
- Версии обновлены до `API 1.4.7` и `Android 1.4.7` (`versionCode = 30`).

## Совместимость

- Android: 8.0 Oreo и новее, `minSdk = 26`.
- Docker image: `ghcr.io/shurshick/shopping-list-api:latest` и `ghcr.io/shurshick/shopping-list-api:v1.4.7`.

## Файлы релиза

- `shopping-list-android-v1.4.7.apk`
- `shopping-list-server-v1.4.7.zip`
- `shopping-list-source-v1.4.7.zip`

## Обновление сервера

1. Сделайте backup БД или выполните `python -m app.cli backup --output backup.json`.
2. Обновите Docker image до `latest` или `v1.4.7`.
3. Перезапустите контейнер API.
4. Проверьте `/health/ready` и `/admin/system`.
