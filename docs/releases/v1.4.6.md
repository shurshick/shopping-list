# Список покупок v1.4.6

Публичный релиз Android-приложения и серверной части для самостоятельного размещения.

## Что изменилось в этом релизе

- На первом экране входа и регистрации Android добавлен чек-бокс "Использовать тестовый сервер".
- При включении чек-бокса автоматически используется `https://rust.bghitech.ru`, а поле ручного адреса сервера становится неактивным.
- В настройках Android прямое переключение сервера заменено на безопасную кнопку "Сменить сервер".
- Перед сменой сервера приложение показывает предупреждение о завершении текущей сессии.
- При подтверждении смены сервера очищаются токен, локальная сессия, кэш списков и очередь офлайн-действий, после чего пользователь возвращается на экран входа.
- Настройка тестового сервера хранится отдельно от ручного адреса, поэтому при отключении чек-бокса приложение возвращается к последнему пользовательскому серверу.
- Исправлено отображение таблицы пользователей в `/admin`: кнопки действий больше не сжимаются в два ряда.
- Версии обновлены до `API 1.4.6` и `Android 1.4.6` (`versionCode = 29`).

## Совместимость

- Android: 8.0 Oreo и новее, `minSdk = 26`.
- Docker image: `ghcr.io/shurshick/shopping-list-api:latest` и `ghcr.io/shurshick/shopping-list-api:v1.4.6`.

## Файлы релиза

- `shopping-list-android-v1.4.6.apk`
- `shopping-list-server-v1.4.6.zip`
- `shopping-list-source-v1.4.6.zip`

## Обновление сервера

1. Сделайте backup БД или выполните `python -m app.cli backup --output backup.json`.
2. Обновите Docker image до `latest` или `v1.4.6`.
3. Перезапустите контейнер API.
4. Проверьте `/health/ready` и `/admin/system`.
