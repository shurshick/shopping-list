# Список покупок v1.4.2

Публичный релиз Android-приложения и серверной части для самостоятельного размещения.

## Что изменилось в этом релизе

- Android-приложение при запуске проверяет последний GitHub Release проекта.
- Если доступна новая APK-версия, вверху главного экрана показывается компактное уведомление.
- Кнопка "Скачать" открывает страницу GitHub Release в браузере.
- Кнопка "Скрыть" больше не показывает уведомление для этой же версии; новая версия появится снова.
- Если GitHub, сеть или сервер обновлений недоступны, приложение ничего не показывает и продолжает работать без ошибок.
- Логика проверки вынесена в отдельные Android-компоненты `UpdateRepository`, `UpdatePreferences`, `UpdateApi` и `VersionComparator`.
- Добавлены JVM-тесты сравнения версий и политики проверки и скрытия обновления.
- Версия Android обновлена до `1.4.2` (`versionCode = 25`).

## Совместимость

- Android: 8.0 Oreo и новее, `minSdk = 26`.
- Docker image: `ghcr.io/shurshick/shopping-list-api:latest` и `ghcr.io/shurshick/shopping-list-api:v1.4.2`.

## Файлы релиза

- `shopping-list-android-v1.4.2.apk`
- `shopping-list-server-v1.4.2.zip`
- `shopping-list-source-v1.4.2.zip`

## Обновление сервера

1. Сделайте backup БД или выполните `python -m app.cli backup --output backup.json`.
2. Обновите Docker image до `latest` или `v1.4.2`.
3. Перезапустите контейнер API.
4. Проверьте `/health/ready` и `/admin/system`.
