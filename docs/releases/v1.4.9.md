# Список покупок v1.4.9

Релиз `v1.4.9` посвящен удобству администрирования сервера. Основной Android-интерфейс не менялся, но версия APK обновлена, чтобы релиз оставался единым.

## Изменения этого релиза

- Админ-панель `/admin` получила общую навигацию: главная, пользователи, списки, приглашения, система, логи и диагностика.
- Таблицы пользователей, списков и приглашений приведены к единому виду, кнопки действий больше не ломают строки таблицы.
- Добавлены поиск, фильтры и сортировка для пользователей.
- Добавлены поиск и фильтры active/archive для списков.
- Добавлены фильтры active/used/expired/revoked и поиск по списку для приглашений.
- Экран системы, логи и диагностика оформлены как отдельные разделы с безопасным копированием отчётов.
- Backend обновлен до `1.4.9`.
- Android обновлен до `1.4.9` (`versionCode = 32`).

## Совместимость

- Старые API endpoints сохранены.
- Android offline queue не менялась.
- Секреты, пароли, raw invite tokens и строки подключения не выводятся в admin/system, logs, diagnostics и metrics.

## Docker image

- `ghcr.io/shurshick/shopping-list-api:latest`
- `ghcr.io/shurshick/shopping-list-api:v1.4.9`

## APK

- `shopping-list-android-v1.4.9.apk`

## Обновление

1. Сделайте backup PostgreSQL или snapshot сервера.
2. Обновите Docker image до `latest` или `v1.4.9`.
3. Проверьте `/health/ready`.
4. Откройте `/admin` и проверьте разделы администрирования.
